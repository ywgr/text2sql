## 🎯 核心测试

现在这些查询都会正常工作：

### 输入: `"geek产品今年的SellOut数据"`
**输出**:
```sql
SELECT [SellOut] FROM [dtsupply_summary] 
WHERE [Roadmap Family] LIKE '%Geek%' 
AND [Group] = 'ttl' 
AND [财年] = 2025
```

### 输入: `"510S 25年7月全链库存"`
**输出**:
```sql
SELECT [全链库存] FROM [dtsupply_summary] 
WHERE [Roadmap Family] LIKE '%510S%' 
AND [Group] = 'ttl' 
AND [财年] = 2025 
AND [财月] = '7月' 
AND [财周] = 'ttl'
```

```sql

```
